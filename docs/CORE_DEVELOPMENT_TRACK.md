# MIYABI コア開発トラック（システム開発）

最終更新: 2026-02-23

## 1. 目的

エンジンとしての基盤機能を段階的に完成させ、2D/3D開発を継続可能にする。

## 2. ステージ定義

### C0: Core Runtime（現状）

- FFI経由でゲーム更新/描画データ受け渡しが成立
- 2D描画/入力/音声/2D物理の最小統合が成立
- SDK v0.1 配布物が成立
- Core Save サブシステム定義が成立（未接続）

### C1: 2D Engine Baseline

- 2D制作向けの実運用機能を提供
- アセットパイプライン、テスト/CI、性能回帰監視を導入
- 2D開発が継続可能な開発基盤を成立

### C2: 3D Foundation

- 透視投影、3Dメッシュ、ライト、深度処理の最小導入
- 2D機能を破壊せず 3D基盤を共存させる

### C3: 3D Engine Baseline

- 3D制作向けの実運用機能を提供
- 表現拡張、性能最適化、3D向けツールを整備

### C4: Ecosystem Competitiveness（長期）

- 開発体験（デバッグ/拡張/ドキュメント）を整備
- マルチプラットフォーム運用と互換性方針を確立

## 3. 現在地

- 現在ステージ: **C0**
- 次ゲート: **C1**
- 直近不足:
  - `find_package` 可能な SDK package config
  - CI（configure/build/smoke）
  - 性能ベースラインと回帰監視
  - Save サブシステムのゲーム実導線統合
  - リンカ警告整理
